World Heat Flow Database Project
=================================

This site documents the configuration, functionality and deployment 
strategy of the `World Heat Flow Database`_ application. It is intended*
to be a comprehensive guide for those unfamiliar with the project in 
order to become quickly up-to-speed with the intentions of 
previous developers.

.. warning:: The application is currently in rapid development so these
   documents can become quickly outdated. If you are changing functionality
   of the application, remember to provide detailed documentation of 
   changes so that those who come after can track what you have done.

.. _World Heat Flow Database: link: http://heatflow.world
